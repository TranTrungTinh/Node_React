# NODE_REACT

Example project using React with server Node include React-Router-v4 + Redux + Firebase

## Installation
```
git clone https://github.com/TranTrungTinh/Node_React.git
``` 

```
cd Node_React
npm/yarn install
``` 

```
cd client
npm/yarn install
``` 

## Usage
```
cd Node_React
npm/yarn run dev
```
